# 🚀 Flask HTOP System Info App

This is a simple **Flask application** hosted on **GitHub Codespaces** that provides real-time system information including:
- ✅ Your **Full Name**
- ✅ **System Username**
- ✅ **Server Time in IST**
- ✅ **Live `top` Command Output**
